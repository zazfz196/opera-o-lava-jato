# Lava Jato Premium

Website profissional de lava-jato com sistema de agendamentos.

## Funcionalidades

- Website responsivo e moderno
- Sistema de agendamentos
- Integração com WhatsApp
- Email de notificação
- Restrições de disponibilidade (domingos apenas, máximo 3 por dia)

## Como Usar

1. Instale as dependências: `npm install`
2. Inicie o servidor: `npm start`
3. Acesse em `http://localhost:3000`

## Deploy

Este projeto está configurado para deploy automático no Render.com.
